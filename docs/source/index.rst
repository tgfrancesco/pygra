PyGRA
=====

**PyGRA** is an interactive scientific data plotter built with Python, PyQt5,
and Matplotlib, inspired by *xmgrace*.

It provides a graphical interface for loading, transforming, fitting, and
visualising multi-column data files — designed for researchers who need fast,
flexible plotting without writing code every time.
